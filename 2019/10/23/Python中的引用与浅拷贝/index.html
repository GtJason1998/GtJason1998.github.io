2. 浅谈变量、对象及引用间的关系

  • 变量 ：变量可以看作是一个标签，它无类型，建立后存入系统变量表
  • 引用 ：引用可以看作是一个特殊的”指针”，它存放着对象的地址
  • 对象 ：对象是计算机分配的一块实际的内存，里边存放着相应的值
